Live Server

With the live server extension in VS Code I can right click my file name in the Explorer and choose open with Live Server. As soon as I hit cmd + s my changes update on the browser.

Chrome Hot Keys:

  • cmd + shift + "t" : opens recently closed windows in chrome broswer even if chrome was accidently quit
  • cmd + opt + "i" : launches the inspector environment to explore the code, dom, and file structure
  • cmd + shift + "[, or ]" : cycles through the open tabs on a browser window
